= lowByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descripción
Extrae el byte de orden inferior (más a la derecha) de una variable (por ejemplo, una palabra).


[float]
=== Sintaxis
`lowByte(x)`


[float]
=== Parámetros
`x`: Un valor de cualquier tipo.

[float]
=== Retorno
`byte`
