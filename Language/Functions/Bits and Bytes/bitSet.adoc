= bitSet()


// OVERVIEW SECTION STARTS

[#overview]
--

[float]
=== Descripción
Establece (escribe un 1) un bit de una variable numérica.


[float]
=== Sintaxis
`bitSet(x, n)`


[float]
=== Parameters
`x`: the numeric variable whose bit to set

`n`: which bit to set, starting at 0 for the least-significant (rightmost) bit

[float]
=== Returns
Nothing
