= highByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descripción
Extrae el byte de orden superior (extremo) de una palabra (o el segundo byte más bajo de un tipo de datos más grande).


[float]
=== Sintaxis
`highByte(x)`


[float]
=== Parámetros
`x`: Un valor de cualquier tipo

[float]
=== Retorna
`byte`
