= bit()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descripción
Calcula el valor del bit especificado (el bit 0 es 1, el bit 1 es 2, el bit 2 es 4, etc.).


[float]
=== Sintaxis
`bit(n)`


[float]
=== Parameters
`n`: the bit whose value to compute

[float]
=== Returns
The value of the bit.
