= bitRead()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descripción
Lee un bit de un número.


[float]
=== Sintaxis
`bitRead(x, n)`


[float]
=== Parámetros
`x`: Número desde el cual leer.

`n`: bit que se leerá, comenzando en 0 para el bit menos significativo (más a la derecha).


[float]
=== Retorna
Valor del bit (0 o 1).
