= bitClear()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descripción
Limpia (escribe un 0) un bit de una variable numérica.


[float]
=== Sintaxis
`bitClear(x, n)`


[float]
=== Parameters
`x`: the numeric variable whose bit to clear

`n`: which bit to clear, starting at 0 for the least-significant (rightmost) bit

[float]
=== Returns
Nothing
